## About

I work at the intersection of data engineering, machine learning, software integration, and practical AI delivery.

My focus is not experimentation for its own sake. I work on systems that need to be reliable, scalable, and useful in real business environments — from data platforms and internal tooling to AI-powered workflows and production-ready applications.

I typically work on problems such as:

- fragmented or unreliable data flows
- manual operational processes that should be automated
- API and system integrations across business-critical workflows
- AI initiatives that need structure, feasibility, and implementation
- internal tools and products that depend on data and intelligent automation
